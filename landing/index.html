Everything you need. Nothing you don't.

A complete local-first stack in three packages.

●

Offline-first

IndexedDB persistence keeps data available with no network. Reads and writes never throw on disconnect.

⇄

Automatic sync

WebSocket relay with exponential backoff reconnect. Operations queue locally and flush when back online.

↻

CRDT conflict resolution

Last-Writer-Wins per field using Hybrid Logical Clocks. Concurrent edits merge correctly — zero application code.

⏱

HLC ordering

Hybrid Logical Clocks give a total order across all clients without a central time authority.

⚙

React hooks

useDoc, useQuery, useMutation built on useSyncExternalStore — concurrent mode safe.

🔒

Secure by default

Token auth, per-client rate limiting, prototype pollution protection, and HLC drift detection.

Simple API. Powerful defaults.

Get started in minutes. No schema migrations, no resolvers, no boilerplate.

store.ts
// npm i @local-first/store
import { createStore, IndexedDBAdapter } from '@local-first/store';

const store = createStore({
  name: 'my-app',
  collections: { todos: {} },
  storage: new IndexedDBAdapter('my-app'),
});

store.sync('ws://localhost:3000');  // one line, auto-reconnects

const todos = store.collection<Todo>('todos');
await todos.put({ id: 't1', text: 'Ship it', done: false });
await todos.update('t1', { done: true });
const done = await todos.find({ done: true });
TodoApp.tsx
// npm i @local-first/react
import { StoreProvider, useQuery, useMutation } from '@local-first/react';

interface Todo { text: string; done: boolean; }

function TodoApp() {
  const todos = useQuery<Todo>('todos');          // reactive, no polling
  const { put, update, del } = useMutation<Todo>('todos');

  return (
    <ul>
      {todos.map(t => (
        <li key={t.id}>
          <input type="checkbox" checked={t.data.done}
            onChange={() => update(t.id, { done: !t.data.done })} />
          {t.data.text}
          <button onClick={() => del(t.id)}>x</button>
        </li>
      ))}
      <button onClick={() => put({ id: crypto.randomUUID(), text: 'New todo', done: false })}>Add</button>
    </ul>
  );
}

export default function App() {
  return <StoreProvider store={store}><TodoApp /></StoreProvider>;
}

How it works

Three simple steps, zero configuration.

01

Write Locally

Every mutation creates an immutable Operation record with an HLC timestamp and is written to IndexedDB immediately. Reads are always local — fast, even offline.

02

Sync Automatically

When online, unsynced operations are pushed to the WebSocket relay. The server deduplicates by operation ID and broadcasts to all other connected clients.

03

Resolve Conflicts

Incoming remote operations are merged using Last-Writer-Wins per field. The operation with the higher HLC wins. Subscribers are notified automatically after merge.
